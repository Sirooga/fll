HOW TO PLAY
Welcome to Tuorla Track! This game is specialy 
created for the 2020 FLL competitions in Tuorla
observatory. This game will require map reading
skills, but not much. Here's how to play the   
game, in better instructions:                  


1) You load the game.                          

2) when the game loaded, you will see a map of 
Tuorla observatory with red marks on it.       

3) By reading the map, go to one of the points 
which is some where in Tuorla.                 

4) If you found the point (slip of paper with a
keyword on it).                                

5) Type in the keyword to the correct point.    

6) If correct, go find the next one!
